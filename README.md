## Project Overview

This repository contains my work for the Software Testing Project in the **Fundamentals of Software Testing** course. The project involved testing a GitLab web application and creating test cases, bug reports, and reflections based on my learning.


## Repository Structure

- **`bug report/`**  
  Contains a sample bug report for an imagined bug.  

- **`images/`**  
  Screenshots of the imagined bug for documentation purposes.  

- **`learning reflections/`**  
  Contains my answers to the learning reflection questions for the course, describing my experience and insights from testing the web app.  

- **`page objects/`**  
  Includes the pages that were tested:  
  - Login page  
  - Registration page  
  - Project page  
  - Issue page  

- **Assignment Instructions (`.md` file)**  
  Contains the instructions provided for the software testing assignment.  


## Notes

This project was completed as part of the **Fundamentals of Software Testing** course. The goal was to practice manual testing, create detailed test cases, explore potential bugs, and reflect on the testing process.
